About the Project | О проекте

DimLifeVPN is a VPN client based on the open-source Hiddify project, adapted and maintained by the DimLife team.

DimLifeVPN — это VPN-клиент, основанный на открытом проекте Hiddify и адаптированный командой DimLife.

---

Reporting Bugs | Сообщение об ошибках

If you encounter a bug or unexpected behavior:

1. Check existing GitHub Issues.
2. Create a new issue if the problem has not been reported.
3. Include screenshots, logs, and steps to reproduce.

Если вы обнаружили ошибку:

1. Проверьте существующие Issues.
2. Создайте новую задачу, если проблема ещё не описана.
3. По возможности приложите скриншоты, логи и шаги воспроизведения.

Development | Разработка

Requirements | Требования

- Flutter 3.38.5+
- Dart SDK 3.10+
- Android SDK

Install Dependencies | Установка зависимостей

flutter pub get

Run Application | Запуск приложения

flutter run

Build APK | Сборка APK

flutter build apk --release

---

Project Scope | Поддерживаемые платформы

At the moment, DimLifeVPN focuses on Android APK builds.

В настоящее время основной платформой проекта является Android (APK).
